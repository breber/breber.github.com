About the Site

I am currently a Computer Engineering student at Iowa State University. I created this website for fun, and to display some of the projects I have worked on.

I created this website from scratch. I find leaning new programming languages interesting. jQuery is a cool language that I recently started playing around with - the menu on the side of the Desktop website view is animated using jQuery. Also, this mobile view is created using jQuery Mobile.
This site is mainly an HTML5, CSS3 and jQuery playground. The majority of the features will not work unless you have a compliant browser. All my testing is done in the latest dev version of Chrome, but feel free to try the latest versions of Firefox and Safari - they should work too for the most part (I guess you could use IE9, because I hear that is mostly HTML5 compliant).
Find me on LinkedIn or Twitter.
Feel free to visit any and all parts of the site. Enjoy!
