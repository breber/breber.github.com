About this Site

I am currently a Computer Engineering student at Iowa State University. I created this website for fun, and to display some of the projects I have worked on.

This website was created from scratch. I was interested in learning about jQuery, so I decided to create a testing website. jQuery is a cool extension for Javascript that I recently started playing around with - the menu on the side is animated using jQuery.

This site is mainly an HTML5, CSS3 and jQuery playground. Some features may not work in older browsers due to compatibility problems with some of the new HTML5 features. I would recommend Google Chrome.
